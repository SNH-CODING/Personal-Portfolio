About The Project

Guest House project was built for Silicon Institute of Technology to manage their guest house booking through a website. It helps their guests or students to book a room easily through a website rather than going to counter for booking. It has a very simple UI design which helps them to access it easily.
It also contains an admin panel which helps the authorities to monitor the booking.

My Responsibility

As a Full Stack Developer, i was responsible for building the entire Website with good UI/UX design. I have build this website in Django Framework with HTML5, CSS3, JavaScript and Bootstrap. For the database i have use MySQL.
It's a fully responsive website means, works properly in both web and mobile view.
It took me around 2 weeks to complete the whole project.
